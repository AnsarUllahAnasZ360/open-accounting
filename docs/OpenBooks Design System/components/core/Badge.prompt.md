Status pill (20px tall) for transaction and report states.

```jsx
<Badge variant="positive" icon="circle-check">Posted</Badge>
<Badge variant="warning" icon="circle-alert">Needs review</Badge>
<Badge variant="ai" icon="sparkles">Categorized by AI</Badge>
```

Use the fixed status vocabulary: Synced (info) · Ready (outline) · Needs review (warning) · Posted (positive) · Match (secondary) · Overdue (negative). `ai` is the only variant for AI provenance — green, never purple.
